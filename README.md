# Pairspire

Pairspire is a simple Swiss-system chess tournament manager for organizers who need rosters, pairings, results, standings, and revision history in one clean workflow.

## Open Source Plan

Stabilize pairing/results with tests, add import/export, document local setup, publish demo data, define contribution labels, and keep the UI organizer-first so clubs can run real events confidently.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to use the app.

## Build

```bash
npm run build
```
